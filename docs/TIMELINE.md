**A 級篩選條件**（2026-04-17 更新：4 維決策框架）：

🔴 **硬線**（任一不符即 skip）：
- HSP 友善文化（面試時避開「打仗 / 開戰場」型語氣）
- 至少 2-3 天 hybrid，或完全 remote
- 薪水 floor NT$1M（絕對底線，不含股票）
- Skills / Network / Flexibility 至少 **1 個 HIGH**

🟡 **可讓**（preference，非必要）：
- B2B SaaS / AI-native 產業
- 薪水 top band（若 S/N/F 有 HIGH 可折讓到 NT$1.1-1.2M）
- Supervisor title 立刻給（1-2 年升可接受）

**4 維 quick check（投遞前 / 面試前問自己）**：
1. 「進去後，我會跟誰學到最多？」（Skills）
2. 「3 年後我離開，履歷 / 人脈是否比現在強？」（Network / Title）
3. 「每天醒來想起這份工作會鬆一口氣還是緊張？」（Flexibility）

至少 1 題 yes → 考慮投 / 接；三題都 no → 薪水 2M 也 pass
